commute(startIdx1=None, endIdx1=None, startIdx2=None, endIdx2=None, assumptions=frozenset())[source]¶

Commute self.operands[startIdx1:endIdx1] with self.operands[startIdx2:endIdx2]. The default, if no indices are provided, is to commute the first operand with the rest (convenient especially when there are just two operands). Returns the equality that equates self to this new version. Give any assumptions necessary to prove that the operands are in Complexes so that the commutation theorem is applicable.
